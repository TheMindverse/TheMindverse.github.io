# Summary

> This is a lower dose than I normally do, switched from hbr to freebase so this would be about the same as doing 310mg of the hbr kind. Still had a fun time rocking out to music, had no CEVS at all but also I didn't throw up which is the first in a long time. I did reach a point where my mind started to feel like it was floating outside of my body, but it never progressed above that. In other trips it starts off like that than ascends into completely disconnected from the world. Considering this was a lower dose I knew I wasn't going to achieve that, but the trip was still fun.
